## Génération d'images placeholder

Pour générer des images placeholder rapidement :

1. Ouvrez `scripts/generate-placeholder-images.html` dans votre navigateur
2. Cliquez sur "Télécharger toutes les images"
3. Déplacez les images téléchargées dans ce dossier (`public/products/`)

## Format recommandé

- Format : JPG, PNG ou WebP
- Taille : 400x400px minimum (ratio 1:1 recommandé)
- Poids : Optimisé pour le web (< 200KB par image)
